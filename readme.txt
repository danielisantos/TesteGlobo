#TesteGloboQA

Testes API - requisições POST, PUT, GET E DELETE

#Pré-requisitos:

1. Plugin do Cucumber for Java
2. Gherkin
3. JDK 11

#Como Executar:

1. Execute a classe Runner com o testng
2. Após a execução, no terminal, clique no link do relatório para visualizar detalhes dos testes executados:
│ View your Cucumber Report at:                                            │
│ https://reports.cucumber.io/reports/10c74d68-300a-401f-ba12-0bc06c595d0e 


#Construído com:

1. Cucumber (BDD)
2. RestAssured
3. cucumber-testng
4. cucumber-junit
